PokéMap

A datamap of all 1025 pokémon, clustered by type, abilities and moves.

Data obtained from PokeAPI .
To search for a specific move or ability, remove spaces. E.g. "Grass Knot" -> "grassknot".
To search for a type but exclude moves with that substring, add a space. E.g. to search "Fire" and exclude "Fire Blast", search for "fire "

Some notes:
1. Move data is taken across all generations, so a move is included for a pokemon if it learned it at any point.
2. Cluster labels were made by manually eyeballing the 'mons in a cluster and picking a fitting word.

The visualization was made with datamapplot, UMAP, and the python library vectorizers.
