PokéMap

A datamap of all 1025 pokémon, clustered by type, abilities, stats and moves.

Data obtained from PokeAPI .

Some notes:
1. Move data is taken across all generations, so a move is included for a pokemon if it learned it at any point.
2. Cluster labels were made by manually eyeballing the 'mons in a cluster and picking a fitting word.
3. Point size is proportional to a 'mon's base stat total.
The visualization was made with datamapplot, UMAP, and the python library vectorizers. To see the code that generated this plot, check out the github repo.
